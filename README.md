# sgtourism
https://sgtourism.herokuapp.com/
A web app for posting Singapore attractions.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
